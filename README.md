# Dotfiles

## git

Change email and name.

```
~/.gitconfig
~/.gitignore
```

## jetbrains

Change the location of IDE directories.

```
~/clion.properties
~/idea.properties
~/phpstorm.properties
~/studio.properties
```

## script

### window_term.sh

Change `app_exec`.

```
$XDG_BIN_HOME/window_term.sh # ~/.local/bin/window_term.sh
```

## shell

```
$MY_CONFIG_SH # $XDG_CONFIG_HOME/mysh
~/.bash_logout
~/.bash_profile
~/.bashrc
~/.pam_environment
~/.profile
~/.xsession
```

## ssh

Fill the template.

```
~/.ssh/config
```

## vim

Create the directory `~/.vim/swap-file/`.

```
~/.vimrc
```
